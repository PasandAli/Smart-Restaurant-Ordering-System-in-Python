# Smart Restaurant Ordering System

[![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white)](https://www.python.org/)  
[![License](https://img.shields.io/badge/License-MIT-green)](https://opensource.org/licenses/MIT)  

A **Python console application** that allows users to place and track restaurant orders. Users can select food items from multiple categories, specify quantities, and view a summary of their order. Perfect for beginners to learn Python programming, handling user input, and working with dictionaries.

---

## Features 

- Select from multiple food categories:
  - **Fast Food:** Burger, Pizza
  - **BBQ:** Tikka, Boti
  - **Chinese:** Chowmein, Fried Rice
- Specify item quantities
- View a summary of all orders
- Continue ordering or exit at any time
- Automatically calculates total price for each item
- Interactive console interface

---

## Full Working & Instructions

The **Smart Restaurant Ordering System** is a console-based Python program that works as follows:

### 1. Start the Program
- Run the Python script `restaurant_ordering.py` in any Python 3 environment.
- The program displays a **welcome message** and the **food categories**.
